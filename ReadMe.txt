========================================================================
    コンソール アプリケーション: AngleEditApplication プロジェクトの概要
========================================================================

[概要]
・Kinectから得られた三次元座標を読み込み、指定した範囲内の角度データを関節分抽出する。

[使い方]
1. 宣言されているinput_position_filenamesのファイル名をプロジェクトのディレクトリに入れておく
2. 抽出する範囲を指定する
   [定数MODEが1の場合]
　・手入力で抽出するフレームの最初のフレーム番号と最後のフレーム番号を入力する
   [2の場合]
  ・input_start_end.datというファイルをプロジェクトのディレクトリに置いておけば、自動で開始フレームと終了フレームを読み込んでくれる。
　・input_start_end.datの書き方
   -----------------------------------
   |(開始フレーム) (終了フレーム)    |
　 |(開始フレーム) (終了フレーム)    |
   |      ・　　　　　　 ・          |
   |　　　・             ・          |
   |　　　・             ・          |
   |(開始フレーム) (終了フレーム)    |
   -----------------------------------
*例外処理はしていないので間違えないでね。

[出力結果]
指定したセットごとに25個の関節の位置データが全て出力される。
